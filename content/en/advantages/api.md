Our internet radio control panel provides an API for almost every function, allowing you to automate tasks and create your own integrations. You can use the API to build scripts for websites, mobile applications, players, and more. The concept is simple: you access a specific panel URL, receive data in JSON format, and use it in scripts written in PHP, JavaScript, or another language. API request authentication is supported.
