title: "Version 3.27.0: Media Manager, Priority Track Queue"
date: 2025-11-23
---

Version 3.27.0 is out:
- Added MediaLibrary section to broadcaster panel
- Introduced Priority Track queue: play selected tracks immediately, ahead of regular playlist
- Fixed memory leaks with FLAC playback
- Fixed playback duration calculation for trimmed files
- Resolved minor issue with winter/summer time switching in scheduler
- Improved performance of Listener Stats
- Upgraded Stereo Tool to version 10.71
- Enhanced radio minisite stability
